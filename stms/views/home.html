What Is STMS?

STMS - String Task Management System is a software that enables doing tasks on letters and words. Students can perform DELETE, DOUBLE, REPLACE, REVERSE, ENCRYPTION and DECRYTION functionalities on words of their choice of maximumly 50 letters. They are able to input these tasks in the Command Line window that this system provides.
